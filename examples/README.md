# examples/

Reserved for concrete worked applications of individual APRs — e.g., a small sample skill specification authored in ASPECT-S form, or a minimal OBSERVE-conformant directory layout for a toy platform.

**Status**: empty by design. Examples will be added only when an APR is stable enough that the example will not need to be torn down and rewritten every revision. Contributions welcome via the process in [`../CONTRIBUTING.md`](../CONTRIBUTING.md).
