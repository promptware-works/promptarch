# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be patient, assume good faith, and focus discussion on the technical merits of the work. Personal attacks, harassment, and discriminatory behaviour have no place here.

## Reporting

Concerns about conduct in this project can be raised by opening a confidential issue on the repository or by emailing the maintainer listed in [`README.md`](README.md). All reports will be reviewed and investigated promptly and fairly.
